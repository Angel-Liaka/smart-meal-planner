# Member 1 Foundations: Expert Systems

## 1. Introduction
Expert systems are computer programs designed to mimic the decision-making ability of a human expert.  
This project introduces a **rule-based meal planning system**, which helps users select meals based on dietary needs and preferences.  
The purpose of this document is to provide the theoretical foundation for the system.

## 2. Background on Expert Systems

### What is an Expert System?
An expert system is a computer application that solves complex problems by reasoning through knowledge, rather than conventional programming.

### Rule-Based Systems
A rule-based system uses **IF–THEN rules** to make decisions. It evaluates conditions (IF) and executes actions (THEN) using a knowledge base.

### Key Components
- **Knowledge Base:** Stores facts and rules relevant to the domain.  
- **Inference Engine:** Processes the rules and derives conclusions.  
- **User Interface:** Allows interaction between the user and the system.  
- **IF–THEN Rules:** Define the logic for decision-making.  
- **Forward vs Backward Chaining (Brief):**  
  - *Forward Chaining:* Starts with known facts and applies rules to infer new facts.  
  - *Backward Chaining:* Starts with a goal and works backward to determine supporting facts.
